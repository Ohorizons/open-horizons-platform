# Azure Infrastructure

This folder is generated by Open Horizons Golden Paths to provide a production-ready starting point for Azure deployment.

## Included Resources

- Log Analytics workspace
- Application Insights component
- Locked-down StorageV2 account

## Deploy

First configure GitHub Actions OIDC from the generated repository:

```bash
./scripts/setup-azure-oidc.sh \
  --resource-group <resource-group> \
  --create-resource-group
```

Then run the `Azure Infrastructure` GitHub Actions workflow in `what-if` or `apply` mode.

For local deployment:

```bash
az deployment group create \
  --resource-group <resource-group> \
  --template-file deploy/azure/main.bicep \
  --parameters name=<component-name> environment=dev location=centralus
```

## Required Access

Use Microsoft Entra workload identity federation or another approved identity with least-privilege access to the target resource group.
